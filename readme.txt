O projeto é composto pela base de dados (1), pela ferramenta de extração de co-ocorrências (2), e pela interface Web. 
A extração (2) ocorre a partir de um corpus processado pela cadeia STRING, que encontra e armazena as co-ocorrências numa base de dados (1),  sendo posteriormente calculadas as diferentes medidas de associação (2). 
A aplicação Web fornece aos utilizadores uma interface que permite explorar esses padrões de co-ocorrências (3).
Em cada um dos módulos desenvolvidos encontra-se a explicação de como cada um destes funciona.
