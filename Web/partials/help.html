The form presented to the user allows him/her to choose the word's lemma (1) and POS (2) and also the association measure (3). However, the user can still choose some additional options like the minimum frequency (5) of each co-occurrence and the maximum number of words (5) in each dependency-property pattern. The button Search (6) shows the result for the selected lemma.

The image shows an example with a set of co-occurrences detected for the noun "carro". The association measure selected for this result was LogDice.

The result is presented in two groups: word on the left and word on the right with whom it co-occurs. For each dependency-property pattern in which the target word is involved, the words that co-occur with the target word are presented in descending order of the selected association measure value. Each word that co-occurs is displayed in the format lemma (l:m), where l is the base 2 logarithm of the co-occurrence's frequency, and m the value of the selected measure. The user may change the association measure without having to re-entering the information about the current word.

In the result, it is possible to get more information about a specific co-occurrence. By clicking a word, it is shown the details of this co-occurrence and the set of sentences that exemplifies the co-occurrence in the corpus.
