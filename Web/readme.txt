A aplicação web está dividida em dois componentes principais, server-side, a parte que é executada no lado do servidor e client-side, a parte que corre no lado do cliente. No lado do servidor, o código executado foi desenvolvido em PHP (pasta server-side) e é responsável pelo acesso à base de dados.deepApp.php -> recebe pedido do clienteDeepWord.php - if($request_type === 'word’) -> pedidos de co-ocorrências para uma determinada palavraDeepConcordance.php - if($request_type === 'concordance') -> pedidos de frases com exemplos para uma determinada co-ocorrência.DeepQueries.php -> pesquisas sql resources/dep_prop.txt -> Padrão dependência-propiedade ,type (pre/post word/verb),relation type(GOVERNOR/GOVERNED),name No lado do cliente, o código foi desenvolvido a partir da framework AngularJS. scripts/deepApp.js -> aplicação angularscripts/services.js -> serviços que ambos os controladores partilham (dados ou funções)scripts/controllers.js -> controladores (um para cada view)styles/ -> css da aplicação partials/-> ficheiro html para cada viewcomponents/ -> diferentes componentes usados (angular, bootstrap).A comunicação entre as duas partes é efectuada através de pedidos assíncronos AJAX (Asynchronous Javascript and XML) através do protocolo HTTP (Hypertext Transfer Protocol), sendo o formato dos dados transmitidos o JSON (JavaScript Object Notation).
